กรอกข้อมูล Token Line

ขั้นตอนการรับ Token Line

การขอรับ Token Line เพื่อที่จะให้ระบบสามารถที่จะสื่อสารกับตัวผู้ใช้ได้โดยตรงเมื่อมีการแจ้งเตือนต่างๆ โดยทางระบบจะอธิบายไปทีละขั้นตอนเพื่อที่จะได้ง่ายต่อความเข้าใจของผู้ใช้ระบบ

เข้าเว็บ Line

ขั้นตอนแรกคือการเข้าไปยังเว็บ Notify Line เป็นเว็บที่ใช้ในการขอ Token Line คลิกเพื่อเข้าสู่เว็บ Notify Line

เข้าสู่บัญชี Line ของผู้ใช้

ขั้นตอนที่สอง จะเป็นการเข้าสู่บัญชีของผู้ใช้งานเพื่อขอรับ Token Line โดยจะกรอก username และ password เพื่อเข้าสู้ระบบ

เข้าไปที่ หน้าของฉัน หรือ My page

ขั้นตอนที่สาม เป็นการเข้าไปที่หน้าที่ชื่อว่า "หน้าของฉัน" เมื่อใช้ภาษาไทย ถ้าเป็นภาษาอังกฤษจะมีคำว่า "My page" เมนูจะแสดงเมื่อคลิกที่ชื่อบัญชีของตัวเองมุมขวาบนของจอภาพ

คลิก ออก Token หรือ Generate Token

ขั้นตอนที่สี่ เมื่อเข้ามาที่หน้า "หน้าของฉัน" หรือ My page แล้วให้เลื่อนหน้าเว็บลงมาจะพบกับปุ่ม "ออก Token" หรือถ้าเป็นภาษาอังกฤษจะใช้คำว่า "Generate Token" ให้คลิกเข้าไป

หน้าต่าง ออก Token หรือ Generate Token

ขั้นตอนที่ห้า เมื่อทำการคลิกปุ่ม "Generate Token" หรือ "ออก Token" แล้วจะเห็นหน้า "Generate Token" หรือถ้าใช้ภาษาไทยเป็น "ออก Token" ขึ้นมา เมื่อมองจากบนลงล่างจะเห็นช่องที่ให้ใส่ชื่อที่เราต้องการที่จะให้มันแสดงเป็นชื่อเมื่อระบบแจ้งเตือน โดยในที่นี้จะใช้คำว่า Smart Feed และเมื่อมองจากช่องใส่ชื่อให้ระบบลงมาอีก จะเจอกับการเลือกบัญชีที่ต้องการให้ระบบส่งแจ้งเตือนให้โดย Notify Line จะออก Token Line ให้แค่กลุ่มและบัญชีของผู้ใช้เท่านั้น เมื่อเลือกเสร็จสิ้นให้ผู้ใช้ทำการคลิกที่ปุ่ม "ออก Token" หรือ "Generate Token" ได้เลย

Token Line

ขั้นตอนสุดท้าย เมื่อทำตามขั้นตอนจากทางด้านบนลงมาครบแล้ว ขั้นตอนนี้ผู้ใช้ก็จะได้ Token Line แล้วและสามารถนำ Token Line ใส่ช่อง Token Line ทางเว็บของระบบ โดยสามารถกดปุ่มลูกศรชี้ขึ้นเพื่อพาผู้ใช้ไปใส่หรือสามารถเลื่อนเมาส์ขึ้นไปใส่ก็ได้เช่นกัน
